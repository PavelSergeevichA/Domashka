# Новый виджет

## Описание:

Проект "Новый виджет" - это приложение на Python для личного кабинета клиента. 
Это виджет, который показывает несколько последних успешных банковских операций клиента. 
На данный момент в проекте реализованы следующие функции:
get_mask_card_number - Маскирует номер карты звездочками, отображаются только первые 6 и последние 4 цифры
get_mask_account - Маскирует номер банковского счета
filter_by_state - Возвращает новый список словарей, содержащий только те словари, у которых ключ
    state соответствует указанному значению
sort_by_date - Возвращает новый список, отсортированный по дате
mask_account_card - Возвращает строку с замаскированным номером карты или счета
get_date - Возвращает строку с датой в формате 'ДД.ММ.ГГГГ'. 
Так же в проекте имеются генераторы: 
filter_by_currency - Возвращает итератор, который поочередно выдает транзакции,
    где валюта операции соответствует заданной (например, USD)
transaction_descriptions - Возвращает описание каждой операции по очереди
card_number_generator - Выдает номера банковских карт в формате XXXX XXXX XXXX XXXX, где X — цифра номера карты
get_transactions_csv - Возвращает список транзакций, загруженный из файла csv
get_transactions_excel - Возвращает список транзакций, загруженный из файла excel

Так же реализован декоратор log, который может логировать работу функции и ее результат как в файл, так и в консоль.
Декоратор log принимает необязательный аргумент filename, который определяет имя файла, в который будут записываться 
логи.Если filename не задан, то логи выводятся в консоль.

## Установка:

1. Клонируйте репозиторий:
```
https://github.com/PavelSergeevichA/Domashka.git
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
## Тестирование:

Проект покрыт тестами pytest. Для их запуска выполните команду:
```
pytest
```
